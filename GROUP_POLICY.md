# Group Policy Baseline

## Purpose

This repository defines the baseline open source governance, security, and collaboration policies for all repositories owned or maintained by **Fenstream Tech Group** and its affiliated GitHub organizations.

## Applicability

This baseline applies to all group repositories by default.

A repository-level exception is permitted only when:

- legally or contractually required
- operationally necessary for the repository’s function
- security-driven and approved by policy owners

Any exception must be:

- explicitly documented in that repository
- approved by designated policy owners
- no less protective than this baseline unless legally required

If no local exception exists, this baseline is authoritative.

## Policy Set

The baseline includes:

- [SECURITY.md](SECURITY.md)

## Ownership and Review

Policy owner: **Group Security and Open Source Governance**

Review cadence: **at least annually** or when legal/security conditions change.
